# AI agents with tool use

When an AI agent calls tools to answer a question, you get a tree of steps: user prompt, tool calls with input/output, and a final response. mdx-notebook captures all of it from a single executable cell.

:::tip
This tutorial calls Google Gemini (2.5 Flash) for real when `GOOGLE_API_KEY` is set. Otherwise it falls back to a mocked agent loop, so the page always renders. Look for the `mocked: true` flag in the result JSON to tell which version you're seeing.
:::

:::
