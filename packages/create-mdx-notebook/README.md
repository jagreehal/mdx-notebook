# create-mdx-notebook

Scaffold a new mdx-notebook tutorial site.

## Usage

```bash
npm create mdx-notebook my-tutorial
# or
pnpm create mdx-notebook my-tutorial
# or
bunx create-mdx-notebook my-tutorial
```

Options:

- `--linked` — keep `workspace:*` dependencies (only useful inside the mdx-notebook monorepo).
- `--no-git` — skip `git init`.
